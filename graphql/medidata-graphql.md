# Medidata Solutions GraphQL API

Medidata provides cloud solutions for clinical trials. The API covers study design, patient data management, EDC data collection, site management, safety events, biomarkers, and clinical analytics for drug development.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Medidata Solutions REST API
